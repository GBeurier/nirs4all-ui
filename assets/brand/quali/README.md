# quali-nirs4all brand assets

Quali-nirs4all variant of the shared nirs4all glyph.

Color: `#4F46E5` (indigo).

Same layout as the ecosystem kit (`../nirs4all`, `../` = ui blue): `icon.svg`, raster icon sizes, `favicon.ico`, horizontal + stacked SVG (+ dark), `og.png`.

Canonical reusable SVG marks are now generated in this repository under
`assets/brands/nirs4all-quality/` by `npm run brand:generate`. This `assets/brand/quali/`
directory is the app-staged raster kit consumed by `nirs4all-quality`; keep it as
a compatibility mirror until that app imports the packaged brand assets directly.
