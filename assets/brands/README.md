# NIRS4ALL reusable brand assets

Generated SVG brand kit for the NIRS4ALL ecosystem.

Run `npm run brand:generate` after editing `scripts/generate-brand-assets.mjs` or the mirrored definitions in `src/brand/index.ts`.

Current brands:
- `nirs4all`: Ecosystem umbrella
- `nirs4all-core`: Portable aggregate runtime
- `nirs4all-ui`: Reusable visual system
- `nirs4all-providers`: Soft-import provider bridge
